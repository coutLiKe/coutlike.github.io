PassGuard — Password Security Analyzer

A browser-based password security tool that calculates Shannon entropy and estimates crack times across multiple attack scenarios — including online throttled attacks, offline bcrypt, MD5, and GPU cluster brute force. Built following NIST SP 800-63B guidelines with a real-time security checklist detecting common passwords, keyboard sequences, and character repetition patterns. Engineered entirely client-side — user passwords are never transmitted or stored.

002
PhishGuard — Phishing URL Detector

A URL threat analysis tool that runs 17 detection checks against real-world phishing techniques documented in MITRE ATT&CK (T1566) — including homoglyph substitution, brand impersonation, subdomain spoofing, open redirect parameters, and high-risk TLD identification. Implements homoglyph normalization to catch typosquatted domains like paypa1.com and a brand-in-subdomain check for patterns like paypal.com.evil.tk. All analysis runs locally — zero network requests or data collection.
